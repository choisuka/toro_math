# 🏰 아르모니아 수학왕국

**아르모니아 프로젝트** — 핀란드·싱가포르·유대인 3개국 교육법 융합 수학 앱

🌐 **라이브**: https://choisuka.github.io/toro_math/

---

## 내용

| 단계 | 이름 | 대상 | 단원 |
|---|---|---|---|
| 1단계 | 수의 근원지 | 초5~중1 | 약수·배수, 소인수분해 |
| 2단계 | 논리의 평원 | 중1~중2 | 방정식, 부등식, 연립방정식 |
| 3단계 | 변화와 관계의 산맥 | 중2~중3 | 일차함수, 이차방정식 |
| 4단계 | 질서와 예측의 바다 | 초5~중1 | 도형, 평균·통계 |
| 5단계 | 고등수학의 설산 | 고1 | 다항식, 복소수, 이차함수 등 |
| 6단계 | 유대인 수학 교육과정 | 초1~중3 | 하브루타, 시험 대비 |

## 학습 방식

각 단원을 **3단계 융합**으로 학습:
- 🇫🇮 **핀란드식** — 직관, 현실 연결
- 🇸🇬 **싱가포르식** — 시각화, 모델
- ✡️ **유대인식** — 하브루타 논리 토론
