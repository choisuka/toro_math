## 수정 내역

### 1. prime-havruta — 중복 카드 삭제
- **문제:** 2번 카드(짧은버전)와 4번 카드(완전판)가 "약수의 개수가 3개인 수의 비밀" 동일 주제 중복
- **조치:** 2번 카드 삭제, 3번→2번, 4번→3번으로 순서 정리 (총 3편)
- **허브 반영:** prime-hub 버튼 "4편 → 3편" 수정

### 2. prime-guide — 신규 세트 추가 (4장)
- **문제:** 소인수분해 개념 가이드가 없어 토론관(prime-havruta) 바로 진입해야 하는 흐름
- **조치:** prime-guide 4장 신규 작성
  - 1/4 🇫🇮 핀란드식 — 블록 해체로 소인수분해 접근
  - 2/4 🇸🇬 싱가포르식 — 수형도(Factor Tree) 시각화
  - 3/4 ✡️ 유대인식 — "왜 소수로만 분해해야 하나?" 질문
  - 4/4 📋 3국 비교 요약 + 약수 개수 공식
- **허브 반영:** prime-hub 맨 위에 "3개국 개념 가이드(4장)" 버튼 추가

### 3. fm-advanced 1/5 — 주기 개념 설명 추가
- **문제:** "주기 = 켜짐 + 꺼짐"이라는 개념이 앞 세트 어디에도 없이 갑자기 등장
- **조치:** 1번 카드에 "주기란?" 박스 추가
  - "주기 = 한 패턴이 반복되는 전체 시간"
  - "처음 상태(켜짐)로 돌아오려면 꺼짐까지 끝나야 한다"

### 4. fm-advanced 4/5 — 소인수분해로 LCM 구하는 공식 설명 추가
- **문제:** "2³ × 3 × 5 = 120" 결론만 나오고 왜 2³인지 설명 없음
- **조치:** 공식 박스 추가
  - "각 소인수의 가장 높은 지수끼리 곱한다"
  - 2는 30에서 2¹, 40에서 2³ → 더 큰 쪽 2³ 선택 등 단계별 설명

### 5. fm-hard10 7/11 — GCD × LCM = A × B 공식 첫 등장 설명 추가
- **문제:** 이 공식이 앞 세트 어디에도 소개되지 않았는데 힌트에 갑자기 등장
- **조치:** "핵심 공식 (처음 배우는 거예요!)" 박스 추가
  - 공식 명시 + 간단한 증명(GCD=g, A=ga, B=gb 형태)
