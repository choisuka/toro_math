🌍 문제 3 — [핀란드] 요금제 경제적 선택
A: 기본 20,000원 + 시간당 3,000원 / B: 시간당 5,500원 (10시간↑ 10% 할인)
🇫🇮 핀란드식 — 구간별 분석

x < 10 (할인 없음)
A < B: 20000+3000x < 5500x → x > 8
→ 8 < x < 10: A 유리 / x ≤ 8: B 유리

x ≥ 10 (10% 할인)
0.9A < 0.9B: (20000+3000x)<5500x → x > 8
→ x ≥ 10에서 항상 A 유리

x < 8: B 유리 / x=8: 동일 / x > 8: A 유리
