−1 < (2x+a)/3 ≤ 5 의 해가 −5 < x ≤ b 일 때, a+b의 값
🇫🇮 핀란드식 — 역산으로 a 찾기
−3 < 2x+a ≤ 15
(−3−a)/2 < x ≤ (15−a)/2

(−3−a)/2 = −5 → a = 7
(15−7)/2 = 4 → b = 4
a + b = 11

⚠️ 헷갈리는 부분
이중부등식은 세 부분을 동시에 같은 연산 적용.
