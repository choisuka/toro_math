⚔️ 심화 8 ★★★ — 역수 부등식
1/x > 2 를 풀어라
🇫🇮 핀란드식 — x의 부호를 모를 때 양변에 x를 바로 곱하면 안 됨.

경우 1: x > 0
1 > 2x → x < 1/2 → 0 < x < 1/2

경우 2: x < 0
1 < 2x → x > 1/2 (x<0과 모순) → 해 없음

최종 해: 0 < x < 1/2

⚠️ 가장 많이 하는 실수
그냥 x < 1/2로 쓰는 것.
x=−1 대입: 1/(−1)=−1 > 2 → 거짓. 반드시 검증.
