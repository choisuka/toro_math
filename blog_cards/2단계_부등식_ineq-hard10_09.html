📚 아르모니아 수학왕국 · 2단계 부등식 · ineq-hard10 9/10
⚔️ 심화 9 ★★★ — 공통해 존재하지 않을 조건
3x−2 > a 이고 2x+1 < 7 의 공통해가 존재하지 않을 조건
🇫🇮 핀란드식
① x > (a+2)/3
② x < 3

공통해 없으려면: (a+2)/3 ≥ 3 → a ≥ 7

✡️ 유대인식 — 경계에서 무슨 일이?
a=7: x > 3 이고 x < 3 → 공통해 없음 ✓
a=6: x > 8/3 이고 x < 3 → 8/3 < x < 3 존재 ✗

경계값을 직접 대입하여 확인하는 것이 가장 확실한 방법.
