⚔️ 심화 10 ★★★ — 최고난도 종합
ax > b의 해가 x < −3일 때, bx > a의 해를 구하여라
🇫🇮 핀란드식 — 부호 분석
해가 x < −3 → a < 0
b/a = −3 → b = −3a → b > 0

bx > a (b>0, 방향 유지)
x > a/b = a/(−3a) = −1/3

🇸🇬 검증 (a=−2, b=6)
ax > b: −2x > 6 → x < −3 ✓
bx > a: 6x > −2 → x > −1/3 ✓

핵심 원리 요약
해의 방향 → 계수 부호 → 새 부등식의 방향 결정.
