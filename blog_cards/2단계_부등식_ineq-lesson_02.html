📘 교안 2/5 — 2교시 대표 풀이 (등호 끝장 토론)
대표 문제: (x−a)/2 ≤ (2x−1)/3 의 자연수 해가 3개일 때 a의 범위
✡️ 등호 끝장 토론 (학생과 함께)

Q: "K가 딱 3이면?" → x < 3, 자연수 1,2뿐 → 2개 (부족)
Q: "K가 딱 4이면?" → x < 4, 자연수 1,2,3 → 3개 맞음 ✓
Q: "K가 3.5이면?" → x < 3.5, 자연수 1,2,3 → 3개 맞음 ✓

→ 3 < K ≤ 4 임을 입으로 소리 내어 읽으며 스스로 깨닫게 함.

핵심 포인트
등호의 포함 여부가 정답을 가른다.
숫자를 직접 대입해서 참/거짓을 확인하는 것이 최고의 치료제.
