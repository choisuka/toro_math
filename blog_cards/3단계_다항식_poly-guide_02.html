🇸🇬 싱가포르식 — 나머지 정리
🇸🇬 싱가포르식 (시각화): f(x)를 (x−a)로 나눈 나머지 = f(a)
나머지 정리: 나눗셈을 직접 하지 않고 대입만으로 나머지를 구하는 마법!

예) f(x) = x³−2x+1을 (x−2)로 나누면
  나머지 = f(2) = 8−4+1 = 5

인수정리: f(a) = 0이면 (x−a)는 f(x)의 인수
  f(1) = 1−2+1 = 0 → (x−1)로 나누어떨어짐

핵심 질문: "왜 (x−a)로 나눈 나머지가 f(a)일까?"
