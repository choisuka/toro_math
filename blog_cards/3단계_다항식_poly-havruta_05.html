🏛️ [단대부고 11번] 복소수의 리듬과 합의 퍼즐
문제: 자연수 n에 대해
f(n) = n × (i+i²+…+iⁿ) × (1+1/i+…+1/iⁿ)
f(k)+f(k+3)=105를 만족하는 자연수 k의 값의 합은?

🇸🇬 싱가포르식 (패턴 발견): "사계절의 합은 0이다"
i는 90도씩 회전: i, −1, −i, 1 → 4개 더하면 0
1/i 도 마찬가지: 4개 더하면 0
→ n이 4의 배수이면 두 괄호 모두 0 → f(n)=0

🇫🇮 핀란드식 (나머지 분석): "살아남은 조각들"
• 나머지 0 (4의 배수): f(n) = 0
• 나머지 1: f(n) = n×(i)×(−i) = n
• 나머지 2: f(n) = n×(i−1)×(−i−1) = 2n
• 나머지 3: f(n) = n×(−1)×(−1) = n
✅ 정답 보기
✡️ 유대인식 하브루타: "105를 만드는 두 짝꿍 찾기"

토로: "k와 k+3은 4로 나눈 나머지가 서로 3 차이 나는 짝꿍이야."

Case 1: k+3이 4의 배수 (k는 나머지 1)
f(k+3)=0 → f(k)=k=105
105÷4 나머지 1 ✓ → k=105

Case 2: k가 나머지 2 (k+3은 나머지 1)
f(k)=2k, f(k+3)=k+3
2k+(k+3)=105 → 3k=102 → k=34
34÷4 나머지 2 ✓ → k=34

최종 답: 105+34 = 139
