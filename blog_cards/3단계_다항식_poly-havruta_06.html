💬 다항식 토론 6/8 — [단대부고] 나머지의 합과 차, 상수 k의 값
🏛️ [단대부고] 나머지의 합과 차로 k를 찾아라
문제: f(x)+g(x)를 [박스]로 나누면 나머지 3, f(x)−g(x)를 나누면 나머지 2.
f(x)+kg(x)가 [박스]를 인수로 가질 때, 상수 k의 값은?

🇫🇮 핀란드식 — "박스 세트: 나머지끼리의 연산"
구조: f+g = [박스]×몫 + 3 → Rf + Rg = 3
구조: f−g = [박스]×몫 + 2 → Rf − Rg = 2
연립: 두 식 더하면 2Rf=5 → Rf=2.5
두 식 빼면 2Rg=1 → Rg=0.5
목표: Rf + k·Rg = 0 → 2.5 + 0.5k = 0 → k=−5

🇸🇬 싱가포르식 — "나머지 막대 그래프"
더했을 때 막대 합 = 3, 뺐을 때 막대 차 = 2
f 두 개의 나머지: 3+2=5 → f 하나당 2.5
g 두 개의 나머지: 3−2=1 → g 하나당 0.5
2.5짜리 막대를 0으로 없애려면 0.5막대가 왼쪽으로 5개 → k=−5

✡️ 유대인식 하브루타: "인수라는 완벽한 약속"

토로: "'인수로 갖는다'는 게 무슨 뜻이야?"
아이: "나머지가 0이 된다는 거요!"
토로: "그럼 f(x)+kg(x)의 나머지는?"
아이: "Rf + k·Rg = 2.5 + 0.5k"
토로: "이걸 0으로 만들면?"
아이: "0.5k = −2.5 → k = −5!"
