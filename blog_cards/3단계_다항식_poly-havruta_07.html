🏛️ [단대부고 11번] 복소수의 리듬 — 3국 완전판
문제: f(n) = n×(i+i²+…+iⁿ)×(1/i+1/i²+…+1/iⁿ)
f(k)+f(k+3)=105를 만족하는 자연수 k의 값의 합은?

🇫🇮 핀란드식 — "4개씩 묶으면 사라지는 마법"
i는 4개마다 제자리: i, −1, −i, 1 → 4개 합=0
1/i도 마찬가지: −i, −1, i, 1 → 4개 합=0
→ n이 4의 배수 → 두 괄호 모두 0 → f(n)=0
n mod 4별 f(n) 값:
• 나머지 0: f(n)=0
• 나머지 1: (i)×(−i)=1 → f(n)=n
• 나머지 2: (i−1)×(−i−1)=2 → f(n)=2n ← 핵심!
• 나머지 3: (−1)×(−1)=1 → f(n)=n

🇸🇬 싱가포르식 — "회전하는 화살표"
i: ↑(위), i²=−1: ←(왼), i³=−i: ↓(아래), i⁴=1: →(오른)
4개 화살표 합 = 0 (원점 복귀)
r=2일 때 중간 괄호 곱 = 2 (대칭 상쇄 후 크기 2)
✅ 정답 보기
✡️ 유대인식 하브루타: "모든 경우를 의심하라"

k를 4로 나눈 나머지 r(k)별 경우의 수:
Case 1: r(k)=1, r(k+3)=0
f(k)=k, f(k+3)=0 → k=105
105÷4=26…1 ✓, 108÷4=27 ✓ → k=105

Case 2: r(k)=2, r(k+3)=1
f(k)=2k, f(k+3)=k+3 → 3k+3=105 → k=34
34÷4=8…2 ✓, 37÷4=9…1 ✓ → k=34

Case 3: r(k)=3, r(k+3)=2
f(k)=k, f(k+3)=2(k+3) → 3k+6=105 → k=33
33÷4=8…1 ≠ 3 ✗ (모순, 탈락)

Case 4: r(k)=0, r(k+3)=3
f(k)=0, f(k+3)=k+3=105 → k=102
102÷4=25…2 ≠ 0 ✗ (모순, 탈락)

최종 답: 105+34 = 139
⚠️ r=2일 때 f(n)=2n (n이 아님!) — 이 차이가 정답을 가릅니다
