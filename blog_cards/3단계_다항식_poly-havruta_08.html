🏛️ [단대부고 11번] 복소수의 리듬과 합의 퍼즐
🇸🇬 싱가포르식 — "사계절의 합은 0이다"
첫 번째 괄호(i의 합): i, −1, −i, 1 → 4개 더하면 0
두 번째 괄호(역수의 합): 1/i=−i 도 마찬가지 → 4개 더하면 0
결론: n이 4의 배수라면 두 괄호 모두 0 → f(n)=0

🇫🇮 핀란드식 — "살아남은 조각들" (n mod 4 분석표)
n (나머지) 첫 번째 괄호 두 번째 괄호 f(n)
4m (0) 0 0 0
4m+1 (1) i −i n
4m+2 (2) i−1 −i−1 2n ← 핵심!
4m+3 (3) −1 −1 n

✡️ 유대인식 하브루타 — "105를 만드는 두 개의 수"
토로: "k와 k+3은 4로 나눈 나머지가 어떨까?"
아이: "하나는 4의 배수이거나, 나머지가 서로 연결되어 있겠네요!"

Case 1: k가 4의 배수 → f(k)=0, f(k+3)=k+3=105 → k=102
102÷4=25 나머지 2 ≠ 0 ✗ 모순!

Case 2: k+3이 4의 배수 (k는 나머지 1) → f(k)=k=105
105÷4=26 나머지 1 ✓ → k=105

Case 3: k가 나머지 2 (k+3은 나머지 1)
2k+(k+3)=105 → 3k=102 → k=34
34÷4=8 나머지 2 ✓ → k=34

🛠️ 미궁 탈출 결과

만족하는 k: 105와 34
최종 합: 105+34 = 139

원장님, 이 문제는 단순한 계산력이 아니라 "복소수의 주기성(4마디)"과 "나머지에 따른 분류"라는 두 가지 핵심 전략을 동시에 요구합니다.
