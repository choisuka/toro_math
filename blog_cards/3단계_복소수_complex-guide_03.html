🌀 복소수 가이드 3/4 — 거듭제곱 미션: 시계 바늘 돌리기
🏛️ [아르모니아 미션] 시간의 지배자: 복소수 거듭제곱 정복
Level 1 — "나침반의 영점 조절" (i의 4마디 리듬)
i¹=i(↑), i²=−1(←), i³=−i(↓), i⁴=1(→) → 4개마다 제자리
질문: "i¹⁰⁰은?" → 100÷4=25…0 → i⁴×²⁵=1 → 정답: 1
유대인식: "길이가 1인 화살표는 아무리 돌려도 반지름 1인 원 위를 벗어날 수 없어. 영원한 순환!"

Level 2 — "45도의 지름길" (1+i)ⁿ
1+i = 45도, 길이 √2
n번 회전 → 45n도, 길이 (√2)ⁿ
(1+i)ⁿ이 처음으로 실수 → 45n이 180의 배수 → n=4 (180도, −4)
(1+i)ⁿ이 처음으로 양의 실수 → 45n이 360의 배수 → n=8 (360도, 16)

Level 3 — "오메가(ω): 120도의 정삼각형"
ω = (−1+√3i)/2 = 120도 화살표
ω³ = 360도 = 1 (제자리 귀환!)
시각화: 360도를 3등분한 피자 조각 3개를 붙이면 한 판(1)이 완성

🛠️ 3초 컷 훈련법
• 실수:허수 = 1:1 → 45도
• 실수:허수 = 1:√3 → 60도 (또는 30도)
• 비율만 보면 문제 90% 해결!
• "합이 4마디마다 0" → 90도짜리(i), "3마디마다 0" → 120도짜리(ω)
