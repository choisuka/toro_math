🌀 복소수 통합 가이드 (5-1~5-5) — 완전 정리
🌀 복소수 완전 가이드 — 5개 테마
━━ 5-1. i의 정체 — "90도 회전하는 핸들" ━━
• −1을 곱하면 180도 뒤집힘 → i는 그 절반, 90도 회전
• 기차(실수)는 앞뒤만. i를 곱하면 핸들을 꺾어 평면 어디든 이동 가능
• 핵심: i를 곱하는 것 ≠ 위로 1칸 이동 (그건 주소). ×i = "90도 돌려라" (명령)
• 증명: i×(a+bi) = −b+ai → 좌표 (a,b)가 (−b,a)로 → 90도 반시계 회전 공식과 일치!

━━ 5-2. 주소 vs 명령 + 1+i=45도 ━━
• 주소: 1+i = 좌표(1,1). 가로1·세로1 → 대각선 → 45도, 크기 √2
• 명령: i(1+i) = (1,1)을 90도 돌리면 (−1,1) = −1+i ← 수식과 그림 일치!
• (1+i)ⁿ: 45도씩 n회전, 크기 (√2)ⁿ

━━ 5-3. 거듭제곱 패턴 ━━
• i의 4마디: i¹⁰⁰ → 100÷4=25 나머지0 → 1
• (1+i)의 8마디: (1+i)⁸=16 (360도 제자리, 크기2⁴)
• ω(120도)의 3마디: ω³=1, ω²+ω+1=0
• 3초 컷: 실수:허수 = 1:1 → 45도 / 1:√3 → 60도

━━ 5-4. 나눗셈 = 각도 빼기 ━━
• (1+i)/(1−i): 45°−(−45°)=90° → i. 5초 컷!
• 전통 계산 확인: (1+i)²/(1+1) = 2i/2 = i ✓
• 곱셈=각도 더하기 / 나눗셈=각도 빼기

━━ 5-5. 수학자↔학생 핵심 Q&A ━━

Q: "(1+i)²=2i라고? 왜 2i야?"
A: "45도 두 번 = 90도. 크기 √2 두 번 = 2. → 크기2, 방향90도 = 2i"

Q: "α³이 실수라는 게 왜 120도랑 관련이야?"
A: "3번 회전해서 실수축 도착 → 3θ=180°×정수 → θ=60°또는 120°"

Q: "나머지2일 때 왜 f(n)=2n이야?"
A: "(i−1)(−i−1) = −i²−i+i+1 = 1+1 = 2 → f(n)=n×2=2n"
