🏛️ [단대부고 14번] 사라진 이차함수의 배꼽을 찾아라!
조건: f(0)=f(6) / |f(−1)|+|f(4)|=0 / −2≤x≤5에서 최댓값 66. f(0)은?

✡️ 토로와의 하브루타
토로: "f(0)=f(6)을 보자마자 뭐가 떠오르니?"
아이: "이차함수의 대칭축이요! 0과 6의 중간인 x=3이 배꼽(축)이겠네요!"

토로: "|f(−1)|+|f(4)|=0은 어떠니? 절대값 두 개를 더해서 0이 되려면?"
아이: "둘 다 0이어야 해요! 즉, f(−1)=0이고 f(4)=0."

토로: "잠깐, 축이 x=3인데 −1이 근이면 대칭점 7도 근이야. 4가 근이면 대칭점 2도 근이고. 이차함수에 근이 4개? 어라?"
아이: "아! 한 그래프가 다른 두 쌍의 근을 동시에 가질 수 없잖아요. 뭔가 모순이에요!"

🔍 핵심 통찰
수식으로 확인: f(x)=a(x−3)²+k로 놓으면
f(−1)=16a+k=0, f(4)=a+k=0
두 식을 빼면: 15a=0 → a=0 → k=0 → f(x)≡0
그러면 최댓값도 0이 되어 66이 될 수 없음. 모순!

⚠️ 원장님 확인 필요
|f(−1)|+|f(4)|=0은 축 x=3과 수학적으로 모순입니다.
원래 조건이 f(−1)+f(4)=0 (절대값 없음)이라면?
→ 16a+k+(a+k)=0 → 17a+2k=0 → k=−17a/2
→ 이 경우 최댓값 조건으로 a, k 모두 결정 가능
✅ 정답 보기
📌 조건 해석
|f(−1)|+|f(4)|=0 → f(−1)=0, f(4)=0이어야 하나 축 x=3과 모순.
시험지 의도: f(−1)+f(4)=0 (이미지 풀이 기준)

풀이
f(x)=a(x−3)²+k (축 x=3)
f(−1)+f(4) = (16a+k)+(a+k) = 17a+2k = 0
→ k = −17a/2

구간 [−2, 5], 축 x=3이 내부에 있음:
• a>0 → 최댓값은 양 끝점 중 큰 쪽
f(−2) = 25a+k = 25a−17a/2 = 33a/2
f(5) = 4a+k = 4a−17a/2 = −9a/2 (음수, 탈락)
→ 33a/2 = 66 → a=4, k=−34

• a<0 → 최댓값은 꼭짓점 f(3)=k=−17a/2=66
→ a=−132/17 (비정수, 탈락)

∴ f(0) = 9(4)+(−34) = 36−34 = 2 　(답: ③)

검산: f(−1)=16(4)−34=30, f(4)=4−34=−30 → f(−1)+f(4)=0 ✓
